704 二分查找 
```
func search(nums []int, target int) int {
    left := 0
    right := len(nums)
    for left < right{
        mid := (left+right)/2
        if nums[mid] < target{
            left = mid + 1
        } else if nums[mid] == target{
            return mid
        } else if nums[mid] > target{
            right = mid
        }
    }
    return -1
}
```
看见有序数组，且无重复元素，需要反应二分法。
个人习惯遵循左闭右开，记忆点是left < right, left更新时为mid+1。

27 移除元素
```
func removeElement(nums []int, val int) int {
    n := len(nums)
    slow := 0
    for fast:= 0;fast < n; fast++{
        if nums[fast] == val{
            continue
        }
        
        nums[slow] = nums[fast]
        slow++
    }
    return slow
}
```
重点是理解数组删除某一元素时，并不是直接抹去就完了，而是内存长度不变，被删除的元素后面所有元素都需要往前挪。
暴力算法即每删除一个后续整体往前挪，而双指针则用slow，fast在O(n)内完成。fast一直往前扫，!=val的搬给slow，slow 往前走；fast 遇到val则忽略，不搬，如此即可通过覆盖的方式删除原值。


977 有序数组的平方 
```
func sortedSquares(nums []int) []int {
    left := 0
    right := len(nums)-1
    res := []int{}
    for left <= right{
        cur1 := nums[left] * nums[left]
        cur2 := nums[right] * nums[right]
        if cur1 >= cur2{
            res = append([]int{cur1},res...)
            left++
        } else {
            res = append([]int{cur2},res...)
            right--
        }
    }
    return res
}
```
注意left < right会漏掉最中间的
