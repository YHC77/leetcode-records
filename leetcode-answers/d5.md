### 242.有效的字母异位词 
```
func isAnagram(s string, t string) bool {
    //针对提问如果输入字符串包含 unicode 字符
    HashBuckect := make(map[rune]int)
    for _,str := range s{
        HashBuckect[str] += 1
    }
    
    for _,str := range t{
        HashBuckect[str] -= 1
    }

    for _,v := range HashBuckect{
        if v != 0{
            return false
        }
    }

    return true
}
```
这题思路就是哈希表法，分别遍历两个字符串，用字典去存储见过的字符，遍历第一个字符串s时记录见过的分别+1，第二个字符串t去查找先前建立好的字典Key。语言方面的知识点主要复习下rune/byte的区别以及range的容器通用型

### 349. 两个数组的交集
```
func intersection(nums1 []int, nums2 []int) []int {
    set := make(map[int]bool)
    for _,num := range nums1{
        set[num] = false
    }

    for _,num := range nums2{
        if _,ok := set[num]; ok{ //知识点错误
            set[num] = true
        }
    }

    res := []int{}
    for key,val := range set{
        if val == true{
            res = append(res,key)
        }
    }
    return res
}
```
和上一题差不多。语法点复习是Go值的存在性不能用值本身去判断。
<details>
<summary>👉Go 将“值”和“存在性”分离，Python 不分离</summary>
习惯性地在判断字典kv是否存在时，使用了if set[nums]{}这种写法。Py 中这样的写法ok因为 Py 允许通过值的 truthiness 隐式判断存在性。但是Go不行，Go 把'存在'也当作返回的信息。
</details>
